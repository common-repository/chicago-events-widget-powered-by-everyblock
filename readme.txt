== Description ==

Add a Chicago Events Widget Powered by Everyblock to a post or page.

== Installation ==

1. Add the folder to wp-content/plugins.
2. On the plugins page in the dashboard, click the "Activate" button under "Chicago Events Widget Powered by Everyblock".
3. Use the shortcode [display_events_widget] to add the widget into any page or post.
